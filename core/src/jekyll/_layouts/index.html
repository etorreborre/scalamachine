Scalaz7 & Scala 2.10 To enforce it's opinion on referential transparency, Scalamachine relies heavily on Scalaz7 for things like typeclasses and monad transformers as well as enumerators/enumeratees for streaming responses. Scalamachine also only runs on Scala 2.10 (currently M7) or newer due to its use of path dependent types and bugs in previous Scala versions.
Webmachine Scalamachine is port of Basho's Webmachine, written in Erlang. Webmachine defines a Flow Diagram that encodes HTTP-semantics into a set of easy to use conventions.
Core scalamachine-core contains the implementation of the Webmachine Flow Diagram, the primary abstraction used in Scalamachine applications -- the resource, a data structure repsenting request and response data, and entry points necessary for implementing support for existing JVM-based HTTP servers.
